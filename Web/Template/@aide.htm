Remarques:

- A cause de restrictions imposées par la sécurité de flash 9, sachez que ce kit graphique aura un comportement optimum quand il sera en ligne sur Internet. Certains effets ne fonctionneront pas de manière optimal lorsque vous consulterez le kit graphique en local sur votre machine (notamment la transition entre les pages) .

- A cause de restrictions imposées par la sécurité de flash 9, vous devez utiliser le navigateur Firefox  si vous souhaiter tester votre site en local sur votre ordinateur. Votre site web devrait apparaître correctement sur les autres navigateur (Internet Explorer, Safari, Opera, Chrome...) lorsque vous mettrez votre site en ligne sur Internet.

- Ce kit graphique est très facile à personnaliser : aucune connaissance de la programmation en flash n'est nécessaire. Les animations flash présentes dans ce kit sont personnalisables par le biais d'une fichier XML et le contenu des pages est constitué de simple (X)HTML.

- The zip file include dfGallery (an imageviewer, http://dezinerfolio.com) and lightbox (http://www.huddletogether.com/projects/lightbox2/).

Comment procéder :

1. Comment personnaliser la page d'accueil?
- Ouvrez le fichier index.htm avec le bloc-note ou n'importe quel éditeur HTML de votre choix et modifier le nom de votre site, le slogan de votre site, l'encart de news, le copyright, etc.
Le fichier index.htm est le fichier principal du kit graphique, cela signifie que le modification que vous ferez dans ce fichier seront visibles sur toutes les pages de votre site web.

2. Comment ajouter une nouvelle page / un nouveau bouton ?
- Ouvez le fichier index.htm avec le bloc-note ou n'importe quel éditeur HTML de votre choix et ajouter le code suivant entre les lignes <!-- Menu --> and <!-- Menu end --> :


myButton('LE NOM DE VOTRE PAGE ','_lenomdemapage.htm');

- Créer un fichier _lenomdemapage.htm avec votre propre contenu HTML. Ne mettez pas les tags <html>,<head>, <body>,... dans ce fichier car ce fichier sera chargé à l'intérieur d'une div dans la page principale.


3. Comment personnaliser les boutons en flash ?
Ouvrez et modifié le fichier button.xml file : Attention à ne PAS ajouter "#" avant les codes couleur.

<?xml version='1.0' encoding='UTF-8' ?>
<button>
<item_default_bgcolor>FFFFFF</item_default_bgcolor>
<item_default_text_color>B9B9B9</item_default_text_color>
<item_mousover_text_color>FFFFFF</item_mousover_text_color>
<item_mousover_bgcolor>669AC6</item_mousover_bgcolor>
<item_default_bullet_color>ffffff</item_default_bullet_color>
<item_mousover_bullet_color>F47B33</item_mousover_bullet_color>
<item_font>Trebuchet MS</item_font><!--utilisez "default" : <item_font>default</item_font> pour un rendu optimal -->
<item_font_size>16</item_font_size>
<item_letter_spaceing>0.2</item_letter_spaceing>
<item_bold>yes</item_bold>
<item_x>19</item_x>
<item_y>2</item_y>
<item_play_sound>yes</item_play_sound>
</button>

4. Comment personnaliser la bannière flash ?
Vous pouvez ajouter à la bannière autant d'images et de slogans que vous le souhaitez, mais sachez trop d'images ralentiront le chargement de votre bannière.
Ouvrez et modifiez le fichier custom.xml et remplacez les valeurs en rouge par les vôtres. Attention à ne PAS ajouter "#" avant les codes couleur.

<?xml version='1.0' encoding='UTF-8' ?>
<header_vars>
<!-- couleur de "la note de musique" permettant d'arrêter ou redémarrer la musique-->
<graphics_color>FFFFFF</graphics_color>
<animation_loop>no</animation_loop>
<!-- temps de transition (en secondes) entre 2 annimations -->
<transition_time_seconds>5</transition_time_seconds>

<header>
<text_header_h1 x_pos="5" y_pos="55" size="19" letter_spacing="0" text_color="FFFFFF">YOUR WAY TO SUCCESS</text_header_h1>
<text_header_h2 x_pos="10" y_pos="80" size="10" letter_spacing="1.9" text_color="FFFFFF">YOUR OWN SLOGANS AND IMAGES</text_header_h2>
<image>images/flashimages/your_image2.jpg</image>
</header>
<header>
.....
</header>
<header>
....
</header>

<!--
Changez play_music="no" si vous ne souhaitez pas ajouter de musique à votre bannière
Changez music_contro="no" si vous ne souhaitez pas ajouter "la note de musique" permettant d'arrêter ou redémarrer la musique
Si vous avez music_file loop="no" vous devez IMPERATIVEMENT mettre music_control="no"
-->
<music_file loop="no" music_control="no" play_music="yes">sounds/music.mp3</music_file>

<!-- vous pouvez remplacer la ligne ci-dessous par <play_transition_sound_file>no</play_transition_sound_file> -->
<play_transition_sound_file>yes</play_transition_sound_file>

<!-- ous pouvez remplacer la ligne ci-dessous par <display_preload graphic_color="669ac6">no</display_preload> si vous ne souhaitez pas afficher le loader de l'annimation-->
<display_preload graphic_color="669ac6">yes</display_preload>


<replay display="yes" x_pos="493" y_pos="135" size="8" letter_spacing="1" bgd_color="FFFFFF" text_color="000000"> REPLAY</replay>
</header_vars>

5. Comment configurer le formulaire de contact ?
Votre formulaire de contact ne fonctionnera que lorsque votre site web sera en ligne et hébergé par un serveur PHP.
Ouvrez le fichier contact.php et modifier la ligne suivante :

$your_email = "example@example.com";

6.comment mettre en place l'effet d'agrandissement des images (lightbox) ?
Ajoutez l'attribut rel="lightbox" aux liens pour activer l'effet lightbox. Par exemple:
<a href="images/image-big-1.jpg" rel="lightbox" title="Optional caption"><img src="images/image-small-1.jpg" alt="my photo" /></a>
